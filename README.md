

  

## Description

  

NestJS app for ABRA Flexi integration

  

## Installation

  

```bash

$  npm  install

```

## Env setup

  
Create .env file with Flexi credentials
```bash

FLEXIBEE_SERVER=

FLEXIBEE_COMPANY=

FLEXIBEE_USERNAME=

FLEXIBEE_PASSWORD=

```
  

## Running the app

  

```bash

$  npm  run  start

```
